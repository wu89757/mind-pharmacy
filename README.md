# 心灵药铺

一个测试网站，可以用几道题目来询问大学生当下的学习生活身心状态，来匹配推荐最适合用户当下阅读的书籍。

## 项目简介

《心灵药铺》是一个基于网页的书籍推荐测试平台，通过问答形式了解用户的心理状态和阅读偏好，运用余弦相似度算法为用户推荐最适合的书籍。

## 功能特点

- 📖 **智能推荐**：基于四维阅读模型（药性、主题、文风、态度）进行书籍匹配
- 🎨 **文艺风格**：采用旧书、信纸风格的视觉设计
- ✨ **动画效果**：逐字显示、淡入淡出等优雅动效
- 📱 **响应式设计**：适配桌面端和移动端

## 技术栈

- HTML5
- CSS3
- JavaScript
- 余弦相似度算法

## 快速开始

### 本地运行

```bash
# 进入项目目录
cd bookweb_prj

# 使用 Python 启动本地服务器
python -m http.server 8080

# 或使用 Node.js
npx http-server -p 8080
```

然后访问 http://localhost:8080

### GitHub Pages 部署

1. 在 GitHub 上创建一个新仓库
2. 将项目文件上传到仓库
3. 进入仓库 Settings → Pages
4. 选择 `main` 分支和 `/ (root)` 目录
5. 点击 Save，等待几分钟后即可访问

## 阅读模型

**四维阅读模型**：
- **药性**：糖（快乐）、药（疗愈）、茶（沉静）、饭（成长）、酒（冲击）
- **主题**：自我、世界、他人
- **文风**：绮丽、清新、直白
- **期待**：理解、鼓舞、陪伴、挑战
